插入数据

```sql
# Insert into 表名（字段列表）values(对应字段列表)
insert into students (name,age) values ('jack',30);
``` 



删除数据

```sql
# 基本语法： delete 表名 set 字段名 = 新值 [where 条件]；
# 高级语法： delete + 表名 + set + 字段 = 值 + [where 条件] + [limit 更新数量];

delete from my_student where age = 30 limit 10;
```     

更新数据 

```sql
# 基本语法： update 表名 set 字段名 = 新值 [where 条件]；
# 高级语法： update + 表名 + set + 字段 = 值 + [where 条件] + [limit 更新数量];
update my_student set name = 'c' where name = 'a' limit 3;
```
