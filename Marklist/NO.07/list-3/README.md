# vscode同步设置
> 实现同步的功能主要依赖于VSCode插件 "Settings Sync"


Setting Sync 快捷键：

1) 上传： `Shift + Alt + U` (Sync: Update / Upload Settings)

2) 下载： `Shift + Alt + D` (Sync: Download  Settings)

###  同步下载设置
`Shift + Alt + U` 

### 同步下载设置
`Shift + Alt + D` 在弹窗里输入你的gist值，稍后片刻便可同步成功
