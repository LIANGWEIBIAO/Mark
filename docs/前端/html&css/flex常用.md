# flex布局

水平居中
```css
.box{ 
    display: flex;
    justify-content:center;//水平居中
}
```

水平居中+垂直居中
```css
.box{ //垂直居中
    display: flex;
    justify-content:center;//水平居中
    align-items:center;
}
```

垂直居中
```css
.box{ //垂直居中
    display: flex;
    align-items:center;
}
```

垂直排列
```css
.box{ //垂直居中
    display: flex;
    flex-direction:column;//垂直排列
}
```
